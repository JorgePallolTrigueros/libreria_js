INSTALAR STABLE DIFFUSION

¿Qué es Stable Diffusion?

Stable Diffusion es una inteligencia artificial capaz de generar imágenes a partir de texto. Es de código abierto y puede instalarse en tu PC para trabajar de forma local, sin depender de servicios en la nube.

Requisitos Previos

  • Tarjeta gráfica compatible con CUDA (NVIDIA recomendada)
  • Python 3.10 instalado
  • Git instalado
  • Windows 10/11 o Linux

Pasos para Instalar AUTOMATIC1111

  1. Clona el repositorio:
    git clone https://github.com/AUTOMATIC1111/stable-diffusion-webui.git
  2. Accede a la carpeta:
    cd stable-diffusion-webui
  3. Ejecuta el script de inicio:
    webui-user.bat (en Windows)
    o
    ./webui.sh (en Linux/Mac)
  4. Se descargarán los modelos automáticamente y al terminar, podrás acceder a la interfaz desde tu navegador en http://localhost:7860.

Consejos Adicionales

  • Instala modelos personalizados copiando archivos .ckpt o .safetensors en la carpeta models/Stable-diffusion.
  • Explora extensiones desde la interfaz web para añadir nuevas funciones.
