Instalación Local (Windows + AUTOMATIC1111)

  1. Descarga e instala Python 3.10.6 desde python.org.
  2. Descarga e instala Git desde git-scm.com.
  3. Ve a GitHub de AUTOMATIC1111.
  4. Haz clic en “Code” > “Download ZIP” o clónalo usando Git.
  5. Descomprime y abre la carpeta.
  6. Ejecuta el archivo webui-user.bat.
  7. Se abrirá una ventana en tu navegador con la interfaz de Stable Diffusion local.
  8. Puedes cargar modelos adicionales (.ckpt, .safetensors) en la carpeta /models/Stable-diffusion.

Desde ahí puedes generar imágenes, usar ControlNet, cargar tus renders, ajustar calidad, estilos y mucho más sin censura.

Uso Online (Web/Colab)

  1. Ve a Hugging Face Spaces.
  2. Busca "Stable Diffusion" y elige una app como diffusers/stable-diffusion.
  3. Sube tu imagen (si quieres usar ControlNet) o escribe un prompt.
  4. Puedes experimentar con estilos, samplers y tamaños.
  5. También puedes usar Google Colab: busca “AUTOMATIC1111 colab” en Google.
  6. Ejecuta las celdas desde arriba hacia abajo, y sigue el link que aparece al final para usar Stable Diffusion sin instalar nada.

Ideal si no tienes una PC potente o quieres probar desde tu navegador sin complicaciones.
