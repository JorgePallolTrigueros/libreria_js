IA de Imagen Local con Mínima Censura

Ejecuta estos modelos en tu propio equipo para tener control total y desactivar cualquier filtro de seguridad que no desees. Necesitarás Python 3.10+, Git y una GPU con CUDA.

1. Stable Diffusion v1.5 (AUTOMATIC1111 WebUI)

Clásico de generación de imagen. Incluye un safety checker que puedes desactivar desde los ajustes o editando modules/sd_safety.py. 


git clone https://github.com/AUTOMATIC1111/stable-diffusion-webui.git
cd stable-diffusion-webui
pip install -r requirements.txt
# Copia tu checkpoint v1.5 a models/Stable-diffusion/
python launch.py

Abre http://localhost:7860 en tu navegador.

2. Stable Diffusion XL (SDXL) Local

Versión avanzada con mayor detalle. El proceso es idéntico a v1.5: basta con usar el checkpoint SDXL. 


# Dentro de la carpeta stable-diffusion-webui:
# Copia tu checkpoint SDXL (e.g., sdxl-base.ckpt) a models/Stable-diffusion/
# Luego:
python launch.py

Accede en http://localhost:7860.

3. Waifu Diffusion v1.3

Modelo afinado en estilo anime, sin filtros NSFW por defecto. 


# Usando AUTOMATIC1111 WebUI:
# Copia waifu-diffusion-v1-3.ckpt a models/Stable-diffusion/
python launch.py

Selecciona el checkpoint “waifu-diffusion-v1-3”.

4. InvokeAI

Implementación alternativa enfocada en usabilidad. No trae filtros automáticos. 


git clone https://github.com/invoke-ai/InvokeAI.git
cd InvokeAI
pip install -r requirements.txt
# Descarga un checkpoint (v1.5 o SDXL) y colócalo en models/
invokeai

Abre http://localhost:9090 para la interfaz.

5. Cualquier Checkpoint + ComfyUI

ComfyUI permite cargar cualquier modelo .ckpt y no incluye filtros de contenido. 


git clone https://github.com/comfyanonymous/ComfyUI.git
cd ComfyUI
pip install -r requirements.txt
# Copia tu .ckpt a models/
python launch.py

Visita http://localhost:8188 para acceder.

Con estas herramientas 100 % locales y open-source, tienes libertad creativa total y control absoluto sobre los filtros.
