IA de Video Instalables Localmente

Las IA locales permiten ejecutar modelos de lenguaje en tu propio equipo, sin subir datos a la nube. Esto ofrece mayor privacidad, control y la posibilidad de personalizar o fine-tunear el modelo. A continuación, dos de las opciones más populares:

5. ComfyUI + AnimateDiff

ComfyUI es una interfaz gráfica open-source para trabajar con modelos de difusión. AnimateDiff es una extensión que añade capacidad de animación frame-by-frame sobre Stable Diffusion. Juntos ofrecen un entorno local sin censura y con flexibilidad total.

9. Deforum

Deforum es una extensión para AUTOMATIC1111 Stable Diffusion WebUI que permite generar animaciones complejas, loops y efectos dinámicos desde tus prompts. Ideal para proyectos de arte generativo y video experimental.

Tutorial de Instalación

Instalar ComfyUI + AnimateDiff

Requisitos previos: Python 3.10+, Git, GPU con CUDA.


# Clona ComfyUI
git clone https://github.com/comfyanonymous/ComfyUI.git
cd ComfyUI

# Instala dependencias
pip install -r requirements.txt

# Clona AnimateDiff en la carpeta de extensiones
mkdir -p extensions
git clone https://github.com/luoshibai/AnimateDiff.git extensions/AnimateDiff

# Inicia la interfaz
python launch.py

Instalar Deforum

Requisitos previos: Python 3.10+, Git, GPU con CUDA, AUTOMATIC1111 WebUI.


# Clona el WebUI de AUTOMATIC1111
git clone https://github.com/AUTOMATIC1111/stable-diffusion-webui.git
cd stable-diffusion-webui

# Instala dependencias
pip install -r requirements.txt

# Clona Deforum Extension
git clone https://github.com/deforum-art/deforum-for-automatic1111.git extensions/deforum

# Inicia el WebUI con Deforum
python launch.py
Tras lanzar cada interfaz, abre http://localhost:8188 (ComfyUI) o http://localhost:7860 (Stable Diffusion WebUI) en tu navegador para empezar a generar.
