🎬 Introducción

iClone es una potente herramienta de animación 3D en tiempo real que te permite crear escenas completas con personajes, iluminación, efectos especiales, cámaras y más. Aquí aprenderás cómo montar una escena paso a paso.

🛠️ Pasos detallados para crear una escena

  1. Abrir iClone: Inicia el programa y selecciona un proyecto nuevo.
  2. Añadir un escenario: Ve a la pestaña Content y arrastra un entorno 3D (bosque, ciudad, estudio, etc.).
  3. Colocar personajes: Agrega personajes desde la biblioteca y ajústalos con las herramientas de movimiento y rotación.
  4. Configurar la cámara: Crea una cámara y ubícala en un ángulo interesante. Puedes usar Preview Cam para exploración rápida.
  5. Añadir iluminación: Usa luces direccionales, ambientales o puntuales para definir el ambiente.
  6. Animar personajes: Usa Motion Puppet o Direct Puppet para aplicar animaciones de forma rápida.
  7. Controlar la línea de tiempo: Ajusta animaciones, expresiones y sincronización con el Timeline.
  8. Añadir FX: Agrega efectos especiales como lluvia, fuego, humo o partículas.
  9. Previsualizar: Reproduce la escena para ver el resultado final antes de renderizar.
  10. Exportar: Renderiza tu escena en video MP4 o como secuencia de imágenes (ideal para postproducción).

📋 Comandos iClone útiles

  // Ejemplo básico de Python Script en iClone (con iClone Python API)
  import RLPy
  
  # Crear una cámara y colocarla
  camera = RLPy.RICamera.Create()
  transform = RLPy.RTransform()
  transform.T().SetXYZ(500, 300, 1000)
  camera.SetTransform(transform)
  
  // Reproducir la animación
  RLPy.RGlobal.Play()
