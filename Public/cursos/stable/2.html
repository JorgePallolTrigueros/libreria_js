🎯 Parte 2: Aplicar ControlNet (OpenPose)

ControlNet es una extensión para Stable Diffusion que permite usar una imagen como “guía” de poses, profundidad, bordes, etc. Ideal para replicar escenas complejas con múltiples personajes como Maldito Viaje.

🧩 Requisitos previos

  • Stable Diffusion Web UI (AUTOMATIC1111)
  • Extensión ControlNet instalada desde Extensions > Available
  • Modelo ControlNet OpenPose (descargado y en la carpeta correcta)
  • Una imagen de referencia de pose (puede ser un sketch, render 3D o frame anterior)

🛠️ Pasos para usar ControlNet con OpenPose

  1. Activa ControlNet en la interfaz de AUTOMATIC1111.
  2. Sube la imagen de referencia en el campo “ControlNet Input”.
  3. Marca Enable y selecciona Preprocessor: openpose y Model: control_v11p_sd15_openpose.
  4. En la parte superior, escribe tu prompt como de costumbre. Asegúrate de que el contenido del prompt coincida con la pose guía.
  5. En Control Weight, deja el valor entre 0.8 y 1.0 para mantener fidelidad a la pose.
  6. Haz clic en “Generate” y observa cómo el personaje sigue exactamente la postura.

📋 Prompt base para probar ControlNet

  morbidly obese sumo wrestler sitting on airplane seat, cinematic light, high detail, 4k, ultra realistic, looking left

⚡ ¿Usas ComfyUI?

En ComfyUI puedes aplicar ControlNet simplemente usando el nodo ControlNetApply y conectando una imagen procesada por OpenPoseDetector. Ideal para automatizar múltiples poses.
